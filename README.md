# Coffee Shop Web Site

Сайт сети кофеен, позволяющий пользователям оформлять заказы, просматривать меню, а администраторам и менеджерам управлять кофейнями и ассортиментом.
### Пользовательские роли
#### Customer
- Регистрация
- Просмотр адресов кофеен, меню
- Оформление заказов
- Просмотр истории заказов

#### Manager
- Управление меню кофейни, менеджером которой он является
- Просмотр заказов своей кофейни
- Аналитика продаж
- Доступ к панели управления осуществляется путем входа через основной сайт:
```
email: manager_karenina@coffee.com
password: manager2
```

#### Admin
- Управление всеми кофейнями, менеджерами и продуктами
- Доступ к панели управления осуществляется путем входа через основной сайт:
```
email: melania@coffee.com
password: adminmel
```
- Управление всеми кофейнями сети
- Управление менеджерами
- Управление продукцией сети кофеен
- Просмотр аналитики:
  - продажи, выручка, заказы по кофейням
  - клиентопоток

## Запуск проекта

```
docker-compose up --build
```

Веб-сайт доступен по ссылке:
https://localhost


Для того чтобы протестировать сайт, лучше добавить данные в БД:
1. Подключение к базе данных PostgreSQL:
```
docker exec -it coffee-postgres psql -U coffee -d coffee_dev
```
2. Вставьте содержимое файла db/insert_data.sql
