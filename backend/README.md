# Coffee Shop API

## Authentication

- **POST** `/api/v1/auth/token`  
  Авторизация: получение токена доступа.

---

## Users - Customer

- **POST** `/api/v1/users/register`  
  Регистрация нового пользователя.
  Создает пользователя с ролью customer и возвращает токен доступа.

  Входные данные:
  ``` 
  "email": "user@example.com",
  "name": "Ivan Ivanov",
  "password": "password" (min: 8 символов)
  ```
- **GET** `/api/v1/users/me`  
  Получение данных текущего пользователя.
- **PUT** `/api/v1/users/me`  
  Обновление данных текущего пользователя.

---

## Products - Public

- **GET** `/api/v1/products/`

  Получение продукта по ID.

## Products - Admin

- **POST** `/api/v1/admin/products/`  
  Создание нового продукта.

  Входные данные: 
  ```
  "name": "Матча латте",
  "description": "Напиток на основе зелёного японского чая матча с молоком",
  "image_url": "https://...",
  "volume": 300,
  "product_type": "non-coffee",
  "price": 250
  ```
- **PUT** `/api/v1/admin/products/{product_id}`  
  Обновление данных о продукте по ID.
- **DELETE** `/api/v1/admin/products/{product_id}`  
  Удаление продукта по ID.

## Orders - Manager

- **GET** `/api/v1/manager/orders/orders`  
  Заказы для кофейни менеджера.
- **GET** `/api/v1/manager/orders/orders-count`  
  Количество заказов для кофейни менеджера за день.





### Аналитика по одной кофейне за выбранный период времени

- **Заказы по дням**  
  Количество заказов за каждый день в выбранном периоде для кофейни менеджера.
- **Заказы по месяцам**  
  Количество заказов, выручка и средний чек по месяцам для кофейни менеджера.
- **Продукция** 
  Топ-продуктов по количеству продаж и выручке для кофейни менеджера.
- **Продажи по времени суток**  
  Группирует заказы по временным интервалам (утро, день, вечер) для кофейни менеджера.

### Аналитика по всем кофейням за выбранный период времени
- **Заказы по дням**  
  Количество заказов за каждый день в выбранном периоде для всех кофеен.
- **Заказы по месяцам**  
  Количество заказов, выручку и средний чек по месяцам для всех кофеен.
- **Рейтинг продуктов**  
  Топ-продуктов по количеству продаж и выручке для всех кофеен.
- **Продажи по времени суток**  
  Количество заказов по временным интервалам (утро, день, вечер) для всех кофеен.
- **Средний чек и количество заказов по кофейням**  
  Средний чек, количество заказов и общая выручка по каждой кофейне.

### Клиентская аналитика за выбранный период времени
- **Топ клиентов**  
  Топ-20 клиентов по количеству покупок, потраченным деньгам и посещённым кофейням.
- **Статистика клиентов**  
  Количество новых и постоянных клиентов.
